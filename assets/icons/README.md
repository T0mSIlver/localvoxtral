# Icon Assets

- `assets/icons/app/AppIcon.png`: source image for the packaged macOS app icon (`.icns` is generated during packaging).
- `assets/icons/menubar/MicIconTemplate.png` and `assets/icons/menubar/MicIconTemplate@2x.png`: primary macOS menu bar icon assets bundled directly with the app.
- `assets/icons/menubar/menubar-icon.pdf` and `assets/icons/menubar/MenubarIcon.svg`: legacy fallback sources if the PNG pair is unavailable.
