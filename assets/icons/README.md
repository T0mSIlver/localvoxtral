# Icon Assets

This folder contains source icons used when packaging `SuperVoxtral.app`.

## Used files

- `assets/icons/app/AppIcon.png`
  Source image for the app icon. The `.icns` file is generated by `scripts/package_app.sh`.
- `assets/icons/menubar/MicIconTemplate.png`
- `assets/icons/menubar/MicIconTemplate@2x.png`
  Menu bar template icon pair.
